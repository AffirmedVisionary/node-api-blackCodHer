Welcome to Affirmed Visionary's API

This page contains some simple ways to get information from a profile file store.

  1. GET/

    Returns this homepage.

  2. GET/api/profiles

    Returns a list of all profiles in profile.json file.

  3. GET/api/profiles/:id

    Returns a single profile based on the id value. Id is an integer between 1 - 4.

  4. POST/api/profiles

    Post a new profile JSON object to server. Will display the list of profiles with the new profile appended.
    Profile must be valid JSON object,e.g:
    {firstname: "jessica", lastname "jones"}

    N.B. Will need an API tester such as Postman to check POST request.

  5. PUT/api/profiles/:id

    Replaces an existing profile object. Will display the list of profiles including the changed object.
    Profile must be valid JSON object,e.g:
    {firstname: "jessica", lastname "jones"}

    N.B. Will need an API tester such as Postman to check PUT request.

  6. PATCH/api/profiles/:id

    Updates an existing profile object. Will display the list of profiles including the updated object. Ignores new object attributes.
    Profile must be valid JSON object,e.g:
    {firstname: "jessica", lastname "jones"}

    N.B. Will need an API tester such as Postman to check PATCH request.

  7. DELETE/api/profiles/:id

    Delete profile by Id.

    N.B. Will need an API tester such as Postman to check DELETE request.
